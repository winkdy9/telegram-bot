# Telegram Bot

## About

Данный проект представляет собой Telegram-бота, реализованного на языке Python.

Цель проекта — изучение работы с Telegram Bot API, HTTP-запросами, асинхронным программированием и парсингом данных с веб-страниц.

В проекте реализованы две версии бота:

* синхронная версия с использованием библиотеки `requests`;
* асинхронная версия с использованием `asyncio` и `aiohttp`.

Асинхронная реализация позволяет выполнять несколько сетевых запросов одновременно и уменьшает время ожидания при работе с внешними сервисами.

---

# Возможности

## Обработка сообщений

Бот умеет:

* получать сообщения пользователей;
* отправлять ответы;
* обрабатывать команды;
* выполнять запросы к внешним API;
* получать данные с веб-сайтов.

---

# Команды бота

## `/quote`

Получение цитаты с сайта:

```
https://quotes.toscrape.com/
```

Пример ответа:

```
Цитата:
"The world as we have created it is a process of our thinking..."
Albert Einstein
```

---

## `/weather`

Получение информации о погоде.

После ввода команды:

```
/weather
```

бот запрашивает название города.

Пример:

```
Пользователь:
Москва

Бот:
Погода в Москва:
Температура: 15°C
Описание: clear sky
```

Для получения информации используется OpenWeather API.

---

## `/headlines`

Асинхронное получение информации из нескольких источников:

* новости о животных;
* последние статьи Habr;
* подборка цитат.

Запросы выполняются параллельно с помощью:

```python
asyncio.gather()
```

Пример:

```
~~~Новости про животных~~~

Статья 1
Статья 2
Статья 3


~~~Новости Habr~~~

Статья 1
Статья 2
Статья 3
```

# Реализация

## Синхронный бот

Файл:

```
sync_bot.py
```

Используемые технологии:

* `requests`
* `BeautifulSoup`

Принцип работы:

```
Telegram API
      |
      |
  requests
      |
      |
 обработка команды
      |
      |
 отправка ответа
```

Каждый запрос выполняется последовательно.

---

## Асинхронный бот

Файл:

```
async_bot.py
```

Используемые технологии:

* `asyncio`
* `aiohttp`
* `BeautifulSoup`

Принцип работы:

```
Telegram API
      |
      |
 aiohttp session
      |
      |
 async обработчики
      |
      |
 отправка ответа
```

Асинхронно выполняются:

* запросы к Telegram API;
* получение новостей;
* получение цитат;
* запрос погоды.

# Работа с API

## Telegram Bot API

Используется для:

* получения обновлений через `getUpdates`;
* отправки сообщений через `sendMessage`.

---

## OpenWeather API

Используется для получения:

* температуры;
* описания погоды;
* информации о городе.
